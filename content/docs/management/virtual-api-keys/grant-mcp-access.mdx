# Grant MCP access

Use this workflow when an application needs governed tool access through Odock MCP endpoints.

<Steps>

<Step>
Open **API Keys**.
</Step>

<Step>
Open the key used by your application or agent.
</Step>

<Step>
Scroll to **MCP Access**.
</Step>

<Step>
Use the MCP access action to add servers.
</Step>

<Step>
Search for the MCP server by name, slug, transport, or id.
</Step>

<Step>
Select one or more MCP servers.
</Step>

<Step>
Confirm the grant.
</Step>

<Step>
Verify the selected servers appear in the MCP Access table.
</Step>

</Steps>
