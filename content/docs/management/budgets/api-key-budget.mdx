# Create an API-key budget

This is the fastest path for application-specific spend control.

<Steps>

<Step>
Open **API Keys**.
</Step>

<Step>
Open the application key.
</Step>

<Step>
Scroll to **Budgets**.
</Step>

<Step>
Create a new budget from that section.
</Step>

<Step>
Confirm **Owner Type** is **API Key** or that the key is selected as the owner.
</Step>

<Step>
Set the schedule, period, timezone, and optional end date.
</Step>

<Step>
Enter the budget amount.
</Step>

<Step>
Keep **Active** checked.
</Step>

<Step>
Save the budget.
</Step>

</Steps>

![Create an apikey scoped budget](/gifs/budgets-apikeys.gif)

## Why Use API-Key Budgets

Use one key and one budget per app, customer, environment, or workload when you need clean spend isolation.
