# Create an API-key quota

This is the fastest path for app-specific usage control.

<Steps>

<Step>
Open **API Keys**.
</Step>

<Step>
Open the application key.
</Step>

<Step>
Scroll to **Quotas**.
</Step>

<Step>
Create a new quota from that section.
</Step>

<Step>
Confirm **Owner Type** is **API Key** or that the key is selected as the owner.
</Step>

<Step>
Choose the metric, period, timezone, and limit.
</Step>

<Step>
Keep **Active** checked.
</Step>

<Step>
Save the quota.
</Step>

<Step>
Run a test request and review the quota window.
</Step>

</Steps>
