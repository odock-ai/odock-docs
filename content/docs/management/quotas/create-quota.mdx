# Create a quota

Use this workflow for organisation, team, user, or API-key usage ceilings.

<Steps>

<Step>
Open your organisation workspace.
</Step>

<Step>
Open **Quotas** from the sidebar.
</Step>

<Step>
Click **Add New Quota**.
</Step>

<Step>
Enter a quota name.
</Step>

<Step>
Choose **Owner Type**: **Organisation**, **Team**, **User**, or **API Key**.
</Step>

<Step>
Select the team, user, or API key when the owner type requires it.
</Step>

<Step>
In **Schedule**, set start date, timezone, period, rollover, and optional end date.
</Step>

<Step>
In **Limits**, choose the metric.
</Step>

<Step>
Enter the numeric limit.
</Step>

<Step>
Keep **Active** checked if the quota should enforce immediately.
</Step>

<Step>
Create the quota.
</Step>

</Steps>
