# Monitor a quota

Quota monitoring connects window-level usage with request-level records.

<Steps>

<Step>
Open **Quotas**.
</Step>

<Step>
Filter by owner type, metric, period, or active state if needed.
</Step>

<Step>
Open the quota.
</Step>

<Step>
Review the **Quota consumption** card.
</Step>

<Step>
Use the previous and next controls to inspect older or newer windows.
</Step>

<Step>
Compare **Used**, **Limit**, and **Remaining**.
</Step>

<Step>
Scroll to **Usage Records** to inspect requests counted against the quota.
</Step>

<Step>
Open a usage record to inspect status, latency, tokens, cost, provider, model, and routing.
</Step>

</Steps>
