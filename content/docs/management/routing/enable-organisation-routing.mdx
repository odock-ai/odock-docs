# Enable organisation routing

The organisation routing switch enables smart routing for the organisation. API-key policies only take effect when this switch is enabled.

<Steps>

<Step>
Open your organisation workspace.
</Step>

<Step>
Open **Settings** from the sidebar.
</Step>

<Step>
Find **Organisation Routing Switch**.
</Step>

<Step>
Turn **Routing enabled** on.
</Step>

<Step>
Wait for the UI confirmation that routing is enabled.
</Step>

</Steps>

![Add a model manually](/gifs/organisation-settings-routing.gif)

## What This Does Not Do

This switch does not create routing policies. After enabling it, configure routing on the API keys that should use routing.
