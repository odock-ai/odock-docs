# Add a model manually

Use manual creation when the catalog does not include the model or when you need custom naming.

<Steps>

<Step>
Open **Models & MCP Servers**.
</Step>

<Step>
Click the create action for a model.
</Step>

<Step>
Enter **Name**. This is the value clients pass in the request.
</Step>

<Step>
Enter **Slug**. This is the upstream provider model id.
</Step>

<Step>
Select **Type**.
</Step>

<Step>
Select capabilities such as vision, reasoning, or tool use.
</Step>

<Step>
Select the **Provider**.
</Step>

<Step>
Select the **Provider Key**.
</Step>

<Step>
Add optional config JSON if your organisation uses custom metadata.

</Step>

<Step>
Create the model.
</Step>

</Steps>
