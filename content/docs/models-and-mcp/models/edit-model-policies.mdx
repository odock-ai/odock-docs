# Edit model policies

<Steps>

<Step>
Open the model detail page.
</Step>

<Step>
Find the **Policies** card.
</Step>

<Step>
Click **Edit**.
</Step>

<Step>
Add IP rules or rate-limit values.


![Edit model policies](/images/pricing-card.png)

</Step>

<Step>
Save policies.
</Step>

<Step>
Test with a virtual API key that has access to the model.
</Step>

</Steps>

For the broader policy model, see [Guardrails](/docs/security-and-guardrails/guardrails).
