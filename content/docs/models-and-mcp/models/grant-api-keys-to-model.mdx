# Grant API keys to a model

Use this workflow when you are starting from the model.

<Steps>

<Step>
Open **Models & MCP Servers**.
</Step>

<Step>
Open the model.
</Step>

<Step>
Find **API Key Access**.
</Step>

<Step>
Click **Batch Add**.
</Step>

<Step>
Filter active keys by type if needed.
</Step>

<Step>
Select one or more API keys.
</Step>

<Step>
Click **Grant Access**.
</Step>

</Steps>
