# Review a model detail page

<Steps>

<Step>
Open **Models**.
</Step>

<Step>
Search or filter the models list.
</Step>

<Step>
Open a model.
</Step>

<Step>
Review the main fields: Name, Slug, Type, Capabilities, Provider ID, and Provider Key ID.
</Step>

<Step>
Review the cards and sections: **Policies**, **Pricing**, **API Key Access**, and **Usage Records**.

</Step>

</Steps>
