# Add models from the catalog

Use the catalog when the model is already listed.

<Steps>

<Step>
Open your organisation in Odock.
</Step>

<Step>
Go to **Models**.
</Step>

<Step>
Click **Batch Create**.

</Step>

<Step>
Select the provider key that Odock should use for these models.
</Step>

<Step>
Search or filter models by name, slug, type, or capability.

</Step>

<Step>
Select the models you want.
</Step>

<Step>
Click **Create Selected Models**.
</Step>

</Steps>

After creation, the models appear in the provider detail page and in **Models & MCP Servers**.
