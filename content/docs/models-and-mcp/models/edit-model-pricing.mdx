# Edit model pricing

<Steps>

<Step>
Open **Models & MCP Servers**.
</Step>

<Step>
Open the model.
</Step>

<Step>
Find the **Pricing** card.
</Step>

<Step>
Click **Edit**.
</Step>

<Step>
Enter the relevant prices in USD per 1M tokens.
</Step>

<Step>
Set cached input discount or embeddings pricing if applicable.

</Step>

<Step>
Save pricing.
</Step>

</Steps>
