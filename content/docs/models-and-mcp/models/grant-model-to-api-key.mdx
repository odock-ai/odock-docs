# Grant a model to an API key

Use this workflow when you are starting from the API key.

<Steps>

<Step>
Open **API Keys**.
</Step>

<Step>
Open the virtual API key used by your application.
</Step>

<Step>
Find **Model Access**.

</Step>

<Step>
Click **Batch Add**.
</Step>

<Step>
Search for models by name, slug, type, provider, or id.
</Step>

<Step>
Select one or more models.
</Step>

<Step>
Click **Grant Model Access**.

</Step>

</Steps>
