# Edit MCP pricing

For pricing fields, calculation, and examples, see [MCP pricing](/docs/models-and-mcp/mcp-servers/pricing).

<Steps>

<Step>
Open the MCP server detail page.
</Step>

<Step>
Find the **MCP Pricing** card.
</Step>

<Step>
Click **Edit**.
</Step>

<Step>
Enter USD per 1M calls, USD per 1M input bytes, and USD per 1M output bytes.

![Edit Mcp Pricing ](/images/mcp-pricing.png)
</Step>

<Step>
Save pricing.
</Step>

</Steps>
