# Edit MCP pricing

<Steps>

<Step>
Open the MCP server detail page.
</Step>

<Step>
Find the **MCP Pricing** card.
</Step>

<Step>
Click **Edit**.
</Step>

<Step>
Enter USD per 1M calls, USD per 1M input bytes, and USD per 1M output bytes.

> Screenshot placeholder: MCP Pricing card in edit mode showing price per 1M calls, input price per 1M bytes, output price per 1M bytes, and the cost formula help text.
</Step>

<Step>
Save pricing.
</Step>

</Steps>
