# Review MCP transport

<Steps>

<Step>
Open **MCP Servers**.
</Step>

<Step>
Open a server.
</Step>

<Step>
Find the **Transport** card.
</Step>

<Step>
Review the transport type.
</Step>

<Step>
Click **Edit** if you need to change endpoint URL for HTTP or SSE, STDIO command, STDIO args, or STDIO environment JSON.

> Screenshot placeholder: MCP server detail page with Transport card in edit mode for HTTP/SSE and a second capture for STDIO fields.
</Step>

<Step>
Save changes.
</Step>

</Steps>
