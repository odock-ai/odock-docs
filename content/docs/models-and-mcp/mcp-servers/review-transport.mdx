# Review MCP transport

For transport behavior and examples, see [Streamable HTTP transport](/docs/models-and-mcp/mcp-servers/streamable-http-transport), [SSE transport](/docs/models-and-mcp/mcp-servers/sse-transport), and [STDIO transport](/docs/models-and-mcp/mcp-servers/stdio-transport).

<Steps>

<Step>
Open **MCP Servers**.
</Step>

<Step>
Open a server.
</Step>

<Step>
Find the **Transport** card.
</Step>

<Step>
Review the transport type.
</Step>

<Step>
Click **Edit** if you need to change endpoint URL for HTTP or SSE, STDIO command, STDIO args, or STDIO environment JSON.

![Edit Mcp Transport ](/images/mcp-transport.png)
</Step>

<Step>
Save changes.
</Step>

</Steps>
