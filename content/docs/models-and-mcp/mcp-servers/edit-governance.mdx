# Edit MCP governance

For the concepts behind allowed tools, blocked tools, semantic filters, policies, and scopes, see [MCP security](/docs/models-and-mcp/mcp-servers/security).

<Steps>

<Step>
Open the MCP server detail page.
</Step>

<Step>
Find the **Governance** card.
</Step>

<Step>
Click **Edit**.
</Step>

<Step>
Enter allowed tool names as a comma-separated list.
</Step>

<Step>
Enter blocked tool names as a comma-separated list.
</Step>

<Step>
Add semantic filter JSON if needed.

![Edit Mcp Gouvernance ](/images/mcp-gouvernance.png)
</Step>

<Step>
Save changes.
</Step>

</Steps>
