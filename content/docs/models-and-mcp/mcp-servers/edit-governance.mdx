# Edit MCP governance

<Steps>

<Step>
Open the MCP server detail page.
</Step>

<Step>
Find the **Governance** card.
</Step>

<Step>
Click **Edit**.
</Step>

<Step>
Enter allowed tool names as a comma-separated list.
</Step>

<Step>
Enter blocked tool names as a comma-separated list.
</Step>

<Step>
Add semantic filter JSON if needed.

> Screenshot placeholder: MCP Governance card showing Allowed Tools, Blocked Tools, Semantic Filter JSON, and Metadata JSON.
</Step>

<Step>
Save changes.
</Step>

</Steps>
