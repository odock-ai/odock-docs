# Grant MCP access to an API key

Use this workflow when starting from the API key.

<Steps>

<Step>
Open **API Keys**.
</Step>

<Step>
Open the virtual API key used by your app or agent.
</Step>

<Step>
Find **MCP Access**.

> Screenshot placeholder: API key detail page with MCP Access section and Batch Add dialog listing MCP servers with transport and auth filters.
</Step>

<Step>
Click **Batch Add**.
</Step>

<Step>
Search for MCP servers by name, slug, id, transport, or auth type.
</Step>

<Step>
Filter by transport or auth if needed.
</Step>

<Step>
Select one or more MCP servers.
</Step>

<Step>
Click **Grant MCP Access**.

> GIF placeholder: Opening an API key, adding MCP Access for a selected MCP server, and seeing the grant appear in the MCP Access table.
</Step>

</Steps>
