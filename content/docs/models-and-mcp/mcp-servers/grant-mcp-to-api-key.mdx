# Grant MCP access to an API key

Use this workflow when starting from the API key.

For how MCP access grants fit with scopes and tool rules, see [MCP security](/docs/models-and-mcp/mcp-servers/security).

<Steps>

<Step>
Open **API Keys**.
</Step>

<Step>
Open the virtual API key used by your app or agent.
</Step>

<Step>
Find **MCP Access**.

</Step>

<Step>
Click **Batch Add**.
</Step>

<Step>
Search for MCP servers by name, slug, id, transport, or auth type.
</Step>

<Step>
Filter by transport or auth if needed.
</Step>

<Step>
Select one or more MCP servers.
</Step>

<Step>
Click **Grant MCP Access**.

</Step>

</Steps>
