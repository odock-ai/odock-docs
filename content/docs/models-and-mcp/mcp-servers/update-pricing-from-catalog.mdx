# Update pricing from the catalog

If an imported MCP server exists in the trusted catalog, catalog pricing can update existing records.

For how MCP pricing is calculated and stored in usage records, see [MCP pricing](/docs/models-and-mcp/mcp-servers/pricing).

<Steps>

<Step>
Open **MCP Servers**.
</Step>

<Step>
Open the trusted catalog batch create dialog.
</Step>

<Step>
Filter to **Existing only** if needed.

</Step>

<Step>
Click **Update Pricing** when available.
</Step>

<Step>
Review the success message.
</Step>

</Steps>
