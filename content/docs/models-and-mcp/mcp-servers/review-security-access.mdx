# Review security and access settings

For auth examples, see [MCP authentication](/docs/models-and-mcp/mcp-servers/authentication). For access grants, scopes, tool rules, semantic filters, and transport-specific concerns, see [MCP security](/docs/models-and-mcp/mcp-servers/security).

<Steps>

<Step>
Open the MCP server detail page.
</Step>

<Step>
Find the **Security & Access** card.
</Step>

<Step>
Review auth type, auth config, OAuth required, enabled status, and public internet flag.

</Step>

<Step>
Click **Edit** if you need to change auth or availability.
![Edit Mcp Security and Access ](/images/mcp-security-auth.png)

</Step>

<Step>
Save changes.
</Step>

</Steps>
