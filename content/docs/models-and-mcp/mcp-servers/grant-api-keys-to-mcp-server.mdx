# Grant API keys to an MCP server

Use this workflow when starting from the MCP server.

<Steps>

<Step>
Open **MCP Servers**.
</Step>

<Step>
Open the MCP server.
</Step>

<Step>
Find **API Key Access**.
</Step>

<Step>
Click **Batch Add**.
</Step>

<Step>
Filter active API keys by key type if needed.
</Step>

<Step>
Select one or more API keys.

> Screenshot placeholder: MCP server detail page with API Key Access table and Batch Add dialog listing active API keys.
</Step>

<Step>
Click **Grant Access**.
</Step>

</Steps>
