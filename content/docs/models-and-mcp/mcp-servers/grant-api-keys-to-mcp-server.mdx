# Grant API keys to an MCP server

Use this workflow when starting from the MCP server.

For how MCP access grants fit with scopes and tool rules, see [MCP security](/docs/models-and-mcp/mcp-servers/security).

<Steps>

<Step>
Open **MCP Servers**.
</Step>

<Step>
Open the MCP server.
</Step>

<Step>
Find **API Key Access**.
</Step>

<Step>
Click **Batch Add**.
</Step>

<Step>
Filter active API keys by key type if needed.
</Step>

<Step>
Select one or more API keys.

</Step>

<Step>
Click **Grant Access**.
</Step>

</Steps>
