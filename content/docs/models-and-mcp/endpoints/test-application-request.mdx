# Test an application request

<Steps>

<Step>
Create or open a virtual API key.
</Step>

<Step>
Grant Model Access for the model.
</Step>

<Step>
Configure the application SDK base URL to the Odock gateway.
</Step>

<Step>
Set the SDK API key to the Odock virtual API key.
</Step>

<Step>
Use the Odock model name in the request.
</Step>

<Step>
Send a small test request.
</Step>

<Step>
Review **Usage Records** in the Odock UI.
</Step>

</Steps>

For a copy-paste quick start, see [Quick Start](/docs/getting-started/quick-start).
