# Test a model from the Playground

Use the Playground to verify a configured model before wiring it into an application.

<Steps>

<Step>
Open your organisation in Odock.
</Step>

<Step>
Go to **AI Playground**.
</Step>

<Step>
Select a configured model.

> Screenshot placeholder: Playground with model selector open and a configured model selected.
</Step>

<Step>
Enter a prompt.
</Step>

<Step>
Send the request.
</Step>

<Step>
Confirm the response succeeds.
</Step>

<Step>
Open **Usage Records** and verify the request appears with model, provider, status, latency, token usage, and cost.

> GIF placeholder: Selecting a model in the Playground, sending a prompt, then opening Usage Records to inspect the resulting usage record.
</Step>

</Steps>
