# Add models from a provider

Once the provider and provider key exist, create models bound to that provider.

<Steps>

<Step>
Open the provider detail page.
</Step>

<Step>
In the **Models** section, choose **Batch Create**.

</Step>

<Step>
Select the provider key.
</Step>

<Step>
Search the model catalog by model name, slug, type, or capability.
</Step>

<Step>
Use filters such as **All models**, **New only**, or **Existing only**.

</Step>

<Step>
Select one or more models.
</Step>

<Step>
Click **Create Selected Models**.

</Step>

</Steps>

![Add Models from catalog](/gifs/providers-add-models.gif
)




The batch create dialog imports model metadata such as type, capabilities, context information, and pricing when the catalog provides it. If a model slug already exists, the UI can create an alternative with the same upstream slug but a different Odock model name.
