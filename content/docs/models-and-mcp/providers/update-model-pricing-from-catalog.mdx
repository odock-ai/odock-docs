# Update model pricing from the catalog

Provider prices change over time. The catalog flow can update pricing for existing model records when catalog pricing is available.

<Steps>

<Step>
Open the provider detail page.
</Step>

<Step>
In the **Models** section, click **Batch Create**.
</Step>

<Step>
Select the provider key if needed.
</Step>

<Step>
If existing catalog models are found, click **Update Pricing (Existing Only)**.

</Step>

<Step>
Review the success message.
</Step>

</Steps>

This updates pricing stored on existing model records. Usage records created after the update use the newer pricing snapshot. Older usage records keep their historical billing data.
