# Edit provider details

Use provider details when you need to adjust the upstream connection.

<Steps>

<Step>
Open **Providers**.
</Step>

<Step>
Click **Configure** on an active provider.
</Step>

<Step>
Review the provider name, type, base URL, timeout, and enabled status.

</Step>

<Step>
Click **Edit** if you need to change the base URL, timeout, enabled flag, or config JSON.
</Step>

<Step>
Save changes.
</Step>

</Steps>
![Edit Provider](/gifs/providers-edit.gif)


Common edits include renaming a provider for an environment, changing a self-hosted inference base URL, increasing timeout for slower models, or disabling a provider when it should not receive traffic.
