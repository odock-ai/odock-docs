# Tools

The organisation UI includes tools for finance review and model testing.

- [Invoices](/docs/tools/invoices) helps you review usage costs and export invoice files.
- [AI Playground](/docs/tools/ai-playground) helps you test a model with an organisation API key.

Invoices only read stored usage data. The AI Playground contacts the runtime only when you test the connection or send a message.
