# Docker Compose

Compose services include:

- `postgres`
- `redis`
- `traefik`
- `server`
- `ui-migrate`
- `ui`
- optional `prometheus`, `alertmanager`, `loki`, `tempo`, `otel-collector`, `promtail`, `grafana`, `node-exporter`, `cadvisor`, `redis-exporter`, and `postgres-exporter`

The server health check calls `/v1/health`.
