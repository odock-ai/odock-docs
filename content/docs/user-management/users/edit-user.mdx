# Edit A User

Role and status changes affect what the person can do in the UI. After changing a user's role, review their team memberships and owned credentials.

<Steps>

<Step>

Open **Users**.

</Step>

<Step>

Click **Open** on the user row.

</Step>

<Step>

Review the profile fields.

</Step>

<Step>

Click the edit action on the detail page.

</Step>

<Step>

Update allowed fields such as name, role, status, or organisation assignment.

</Step>

<Step>

Click **Save Changes**.

</Step>

<Step>

Confirm the updated values appear on the user detail page.

</Step>

</Steps>
