# Add a New User

Some organizations use invitation-only onboarding. Others also allow direct user creation from the **Users Table**.

Use direct creation when the account should exist immediately in the workspace. Use invitation enrollment when the user should activate access themselves and confirm their profile.

<Steps>

<Step>

Open **Users**.

</Step>

<Step>

Stay on the **Users Table** tab.

</Step>

<Step>

Click **Invite User**.

</Step>

<Step>

Pre-fill the form with the user’s information, such as **Name** and **Email**.

</Step>

<Step>

Choose the user’s **Role**.

</Step>

<Step>

An invitation will be sent to the user’s email address.

</Step>

<Step>

Submit the form and confirm that the user appears in the table.

</Step>

<Step>

The user will become active after confirming the invitation and verifying their information.

</Step>

</Steps>
