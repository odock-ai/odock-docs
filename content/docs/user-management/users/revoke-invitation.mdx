# Revoke An Invitation

Revoke an invitation when it was sent to the wrong person or access should no longer be granted.

<Steps>

<Step>

Open **Users**.

</Step>

<Step>

Open the **Enrollment** tab.

</Step>

<Step>

Find the pending invitation.

</Step>

<Step>

Use the **Revoke** action.

</Step>

<Step>

Confirm the revocation in the dialog.

</Step>

<Step>

Confirm the invitation status changes and the old link can no longer be used.

</Step>

</Steps>
