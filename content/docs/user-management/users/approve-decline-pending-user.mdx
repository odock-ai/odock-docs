# Approve Or Decline A Pending User

Use this workflow when a user appears in the table with `PENDING` status.

<Steps>

<Step>

Open **Users**.

</Step>

<Step>

Filter **Status** to `PENDING`.

</Step>

<Step>

Review the user row.

</Step>

<Step>

Click **Approve** to set the user active, or **Decline** to revoke the pending access.

</Step>

<Step>

Confirm the action in the dialog.

</Step>

<Step>

Refresh the table and verify the status changed.

</Step>

</Steps>
