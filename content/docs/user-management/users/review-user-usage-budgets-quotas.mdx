# Review User Usage, Budgets, And Quotas

Use this workflow when investigating a user's traffic, spend, or limits.

For details, see [Usage Monitoring](/docs/observability/usage-records), [Budgets](/docs/management/budgets), and [Quotas](/docs/management/quotas).

<Steps>

<Step>

Open the user detail page.

</Step>

<Step>

Review **Usage Records** to understand recent gateway traffic attributed to the user.

</Step>

<Step>

Review **Budgets** to understand spending limits connected to the user.

</Step>

<Step>

Review **Quotas** to understand volume or token limits connected to the user.

</Step>

<Step>

Follow the linked records if you need to adjust a limit or investigate a request.

</Step>

</Steps>
