# Resend An Invitation

Use resend when the email was missed, the link expired, or the user needs a fresh invitation path.

<Steps>

<Step>

Open **Users**.

</Step>

<Step>

Open the **Enrollment** tab.

</Step>

<Step>

Find the invitation in the **Invitations** table.

</Step>

<Step>

Use the **Resend** action on the invitation row.

</Step>

<Step>

Confirm the invitation remains trackable in the table.

</Step>

</Steps>
