# Review Users

Use the **Users Table** for periodic access review. Look for users with broad roles, stale memberships, or revoked accounts that still own credentials.

<Steps>

<Step>

Open **Users**.

</Step>

<Step>

Stay on the **Users Table** tab.

</Step>

<Step>

Use search to find a person by name.

</Step>

<Step>

Use **Role** or **Status** filters to narrow the list.

</Step>

<Step>

Click **Open** on a user row to inspect details.

</Step>

</Steps>
