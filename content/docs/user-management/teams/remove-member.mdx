# Remove A Member

Removing a member removes that team relationship. It does not automatically remove the user from the organisation, and it does not automatically revoke API keys. Review user-owned and team-owned keys if the membership change is part of an offboarding process.

<Steps>

<Step>

Open the team detail page.

</Step>

<Step>

Find the user in the **Members** section.

</Step>

<Step>

Click **Remove**.

</Step>

<Step>

Confirm the removal in the dialog.

</Step>

<Step>

Refresh the section if needed and confirm the user is no longer listed.

</Step>

</Steps>
