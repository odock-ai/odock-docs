# Review Team-Owned API Keys

Team-scoped API keys are virtual keys owned by the team. Use them for services, agents, or workflows that belong to the group rather than one person.

For details, see [Virtual API Keys](/docs/management/virtual-api-keys), [Models](/docs/models-and-mcp/models), and [MCP Servers](/docs/models-and-mcp/mcp-servers).

<Steps>

<Step>

Open the team detail page.

</Step>

<Step>

Find the **API Keys** section.

</Step>

<Step>

Review the keys owned by the team.

</Step>

<Step>

Open a key that needs inspection.

</Step>

<Step>

Check its status, expiry, model access, MCP access, usage, budgets, and quotas.

</Step>

<Step>

Create team scoped virtual api key
</Step>

</Steps>

Review team keys whenever membership changes, the team starts a new workflow, or usage grows unexpectedly.
