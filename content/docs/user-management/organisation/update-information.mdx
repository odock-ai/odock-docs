# Update Organisation Information

Use this workflow when the organisation display name or operational contact needs to change.

<Steps>

<Step>

Open your organisation workspace.

</Step>

<Step>

In the sidebar, click **Settings**.

</Step>

<Step>

Review the current organisation name and contact.

</Step>

<Step>

Click the edit action on the settings detail page.

</Step>

<Step>

Update **Name** or **Contact**.

</Step>

<Step>

Click **Save changes**.

</Step>

<Step>

Confirm that the updated values appear on the settings page.

</Step>

</Steps>

Use the organisation name for a human-readable workspace label. Use the contact field for the team or person responsible for operational coordination.
