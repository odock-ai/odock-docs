# Enable Routing

The routing switch in organisation settings enables or disables smart routing for the organisation. It does not define the full routing behavior by itself. Routing policies are configured on virtual API keys, and candidate models must still be configured and accessible.

For the full routing concept and setup, see [Routing](/docs/management/routing).

<Steps>

<Step>

Open your organisation workspace.

</Step>

<Step>

In the sidebar, click **Settings**.

</Step>

<Step>

Find the routing settings card.

</Step>

<Step>

Turn routing on when the organisation is ready to let eligible API keys use smart routing policies.

</Step>

<Step>

Keep routing off when smart routing should be prevented globally for the organisation.

</Step>

<Step>

Review the virtual API keys that have routing policies configured.

</Step>

</Steps>
